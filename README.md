## 📋 What is this?

**My-Progress** is a fully offline, browser-based progress tracker 
for a structured 6-month journey to become a production-ready 
**AI/ML Engineer** — tailored for students who already have 
Python and ML foundations and want to excel.

Built to leverage:
- 🎓 **GitHub Student Developer Pack** ($200K+ free tools)
- ☁️ **Oracle Cloud** (4 free AI certifications)
- 🆓 **100% free** — no backend, no database, runs in your browser

| Feature | Description |

| 📊 **Dashboard** | KPIs, charts, activity heatmap, milestones |
